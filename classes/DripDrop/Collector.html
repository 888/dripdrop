Class DripDrop::Collector
In: lib/dripdrop/collector.rb
Parent: Object

Collector is meant to be subclassed. It‘s used to provide basic pub/sub functionality. Subclasses should provide on_recv, which gets called on receipt of a message. If pub_addr is specified publish can be called, which sends a message to the pub socket.

    def initialize(sub_addr='tcp://127.0.0.1:2900',pub_addr=nil)
      sub_addr_uri = URI.parse(sub_addr)
      host, port   = sub_addr_uri.host, sub_addr_uri.port.to_i
      scheme       = sub_addr_uri.scheme.to_sym
      @sub_addr    = ZM::Address.new(host, port, scheme)
      @sub_reactor = ZM::Reactor.new(:sub_reactor)
      
      if @pub_addr
        @pub_addr  = URI.parse(pub_addr)
        @publisher = CollectorPub.new(@pub_addr)
      end
    end
